这道题给了两个 Jar 包：Heritage-app.jar 和 Heritage-gw.jar

Heritage-gw.jar

@Component
public class SecurityFilter implements GlobalFilter {
    private static final Logger log = LoggerFactory.getLogger(SecurityFilter.class);
    private static final List<Pattern> ATTACK_PATTERNS = Arrays.asList(Pattern.compile("<script>(.*?)</script>", 2), Pattern.compile("src[\r\n]*=[\r\n]*\\'(.*?)\\'", 42), Pattern.compile("</script>", 2), Pattern.compile("<script(.*?)>", 42), Pattern.compile("eval\\((.*?)\\)", 42), Pattern.compile("expression\\((.*?)\\)", 42), Pattern.compile("javascript:", 2), Pattern.compile("vbscript:", 2), Pattern.compile("onload(.*?)=", 42), Pattern.compile("include\\s+(.*?)", 42), Pattern.compile("require\\s+(.*?)", 42), Pattern.compile("support", 2), Pattern.compile("runtime", 2), Pattern.compile("processbuilder", 2), Pattern.compile("scriptenginemanager", 2), Pattern.compile("eval", 2), Pattern.compile("exec", 2), Pattern.compile("yaml", 2), Pattern.compile("tostring", 2), Pattern.compile("character", 2), Pattern.compile("substr", 2), Pattern.compile("getmethods", 2), Pattern.compile("invoke", 2), Pattern.compile("\\.\\./", 2), Pattern.compile("\\.\\.\\\\", 2), Pattern.compile("%2e", 2), Pattern.compile("%2f", 2), Pattern.compile("/[^/]*/internal/", 2));

    public SecurityFilter() {
    }

    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {
        ServerHttpRequest request = exchange.getRequest();
        String queryParams = request.getURI().toString();
        if (queryParams != null && this.isAttackString(queryParams)) {
            ServerHttpResponse response = exchange.getResponse();
            response.setStatusCode(HttpStatus.BAD_REQUEST);
            DataBuffer buffer = response.bufferFactory().wrap("Invalid request".getBytes(StandardCharsets.UTF_8));
            return response.writeWith(Flux.just(buffer));
        } else {
            return "POST".equalsIgnoreCase(request.getMethodValue()) && request.getHeaders().getContentType() != null && request.getHeaders().getContentType().includes(MediaType.APPLICATION_JSON) ? DataBufferUtils.join(request.getBody()).flatMap((dataBuffer) -> {
                byte[] bytes = new byte[dataBuffer.readableByteCount()];
                dataBuffer.read(bytes);
                DataBufferUtils.release(dataBuffer);
                String requestBody = (new String(bytes, StandardCharsets.UTF_8)).toLowerCase();
                if (this.isAttackString(requestBody)) {
                    exchange.getResponse().setStatusCode(HttpStatus.BAD_REQUEST);
                    return exchange.getResponse().setComplete();
                } else {
                    final DataBuffer buffer = exchange.getResponse().bufferFactory().wrap(bytes);
                    ServerHttpRequest mutatedRequest = new ServerHttpRequestDecorator(request) {
                        public Flux<DataBuffer> getBody() {
                            return Flux.just(buffer);
                        }
                    };
                    return chain.filter(exchange.mutate().request(mutatedRequest).build());
                }
            }) : chain.filter(exchange);
        }
    }

    private boolean isAttackString(String queryParams) {
        return ATTACK_PATTERNS.stream().anyMatch((pattern) -> {
            return pattern.matcher(queryParams).find();
        });
    }

securityFilter 是核心类，其主要目的就是限制请求 /api/internal 路径以及过滤危险的请求内容，比如 runtime、processbuilder、scriptenginemanager 等。同时如果要以 POST 请求资源，Content-Type 需要设置成 application/json。

不过后来注意到这里定义了一个 Annotation类

ValidateName

@Target({ElementType.FIELD, ElementType.PARAMETER})
@Retention(RetentionPolicy.RUNTIME)
@Documented
@Constraint(
    validatedBy = {NameValidator.class}
)
public @interface ValidateName {
    String message() default "Invalid name";

    Class<?>[] groups() default {};

    Class<? extends Payload>[] payload() default {};
}

NameValidator

public class NameValidator implements ConstraintValidator<ValidateName, String> {
    private static final String REGEX_NAME = "^[a-f\\d]{32}$";
    private static final Pattern PATTERN = Pattern.compile("^[a-f\\d]{32}$");

    public NameValidator() {
    }

    public boolean isValid(String value, ConstraintValidatorContext context) {
        if (!StringUtils.isEmpty(value) && !PATTERN.matcher(value).matches()) {
            context.buildConstraintViolationWithTemplate(String.format("%s", value)).addConstraintViolation();
            return false;
        } else {
            return true;
        }
    }
}

限制了 name 只能为32位的字母数字组合
